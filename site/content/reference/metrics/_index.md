# Metrics Reference

Definitions and formulas for every metric gh-velocity computes.

- **[Lead Time]({{< relref "lead-time" >}})** — Issue created to issue closed
- **[Cycle Time]({{< relref "cycle-time" >}})** — Work started to completed (two strategies)
- **[Velocity]({{< relref "velocity" >}})** — Effort completed per iteration
- **[Throughput]({{< relref "throughput" >}})** — Items completed in a time window
- **[Quality]({{< relref "quality" >}})** — Defect rate, hotfix detection, category composition
